### How It Works

1. **Fetch** - Automated workflow pulls the latest articles from all sources
2. **Filter** - Keyword matching identifies relevant articles
3. **Deduplicate** - Fuzzy matching groups similar stories across sources
4. **Summarize** - AI generates concise summaries and trend analysis
5. **Publish** - Posts are automatically deployed to GitHub Pages
