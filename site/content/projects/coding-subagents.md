coding-subagents is a library of 127 production-ready AI agents spanning core development, language specialists, infrastructure, security, data/ML, and more — packaged for both Claude Code and GitHub Copilot so you get domain-expert guidance instead of generic AI help. The vision is to make every developer's AI assistant as knowledgeable as a specialist by providing plug-and-play agents that encode real-world best practices.
